Replay benchmark on 20 real Perplexity Deep Research examples captured in April 2026. After cleanup and normalization, the slice contains 76 valid phase-3 decision points.

  • Source data: real Perplexity Deep Research outputs captured as JSON, then cleaned into valid re-query states.
  • Each row is one historical phase-3 re-query state. The primary decision being scored is just: search more vs converge.
  • A correct action means matching the labeled stop/continue choice for that exact historical state.
  • Primary safety metric: false-converge rate (premature converge with missing required sub-topic evidence).
  • Three-lane replay: the same bounded state is replayed across raw 9B, 9B + Memla, and Claude raw.
  • Economics shown here use a public Perplexity-style scenario: $2/M input, $8/M output, plus a modeled $0.005 search-equivalent fixed cost per frontier decision.
  • Memla lane assumes local qwen3.5:9b routing at a modeled $0.002 fixed cost per decision, with fallback priced at the frontier lane.
  • Observed replay fallback proxy is 13.16%, using verifier overrides as the escalation-rate assumption.
  • Treat the dollar figure as a pre-pilot model. The live pilot should recalculate economics against customer-internal costs inside the VPC.

def phase3_decide(context, sub_question):
    result = route_decision(
        context=context,
        candidates=["search_more", "converge"],
        verifier="coverage_check",
        # escalates to frontier if confidence below threshold
        escalation_threshold=0.85
    )
    return result.decision
