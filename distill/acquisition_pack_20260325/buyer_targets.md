# Target Buyers

## Highest Fit

- LangChain: agent engineering and memory infrastructure; Memla can become the compounding coding-intelligence layer.
- Mem0: direct memory-layer adjacency; Memla extends memory into transferable coding behavior.
- Coding assistant / agent companies: a way to turn model usage into owned repo intelligence instead of just better prompting.

## Strong Strategic Interest

- Developer workflow platforms that already own IDE or CLI surface area.
- Agent infrastructure teams that need retention, personalization, and compounding task performance.
- Enterprise AI tool builders who want org-specific coding memory without rebuilding the layer from scratch.

## Best Initial Pitch

Do not pitch 'we cloned Claude.'

Pitch:
- Memla turns accepted coding work into transferable intelligence.
- Memla uses transmutations, not just embeddings.
- Memla materially improves second-repo coding guidance in front of the same frontier model.
