Memla turns frontier-model coding usage into transferable intelligence: it reaches 1/1 on the home-repo coding holdout, improves empty-memory second-repo planning from 0.6111 to 0.8611 file recall and from 0 to 1 command recall, and shifts a Claude head-to-head from 0.1667 to 0.9167 file recall on unseen second-repo tasks.
